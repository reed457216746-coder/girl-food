# 家里吃更香

一个给 iPhone 使用的轻量网页/PWA，用来记录重庆外食花费、重口味比例和每周/月统计。

## 功能

- 快速记录一顿外食：日期、类型、花费、地点、辣度、口味、堂食/外卖、备注
- 本周和本月统计：外食次数、花费、重口味占比、外卖次数、最高一顿、预计可省金额
- 最近外食列表，支持编辑和删除
- 本地数据保存到浏览器 `localStorage`
- JSON 备份导出/导入
- CSV 导出
- PWA 支持：可添加到 iPhone 主屏，已缓存核心文件

## 使用

直接打开 `index.html`，或发布到 GitHub Pages 后用 Safari 打开网址。

GitHub Pages 地址：

https://reed457216746-coder.github.io/girl-food/

iPhone 上推荐：

1. 用 Safari 打开网页
2. 点击分享按钮
3. 选择“添加到主屏幕”

## 备份提醒

数据存在当前浏览器本地。清理浏览器数据、换手机、切换浏览器都可能导致记录不可见。建议每周使用“备份”导出 JSON 文件。
